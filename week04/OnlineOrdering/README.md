# Online Ordering System

A **C# console project** that simulates an online ordering system.  
It handles **customers, addresses, products, and orders** with packing/shipping labels and total cost calculation.

##  Project Structure
- `Address.cs` ‚Äì Represents a shipping address and checks if in USA.
- `Customer.cs` ‚Äì Stores customer details.
- `Product.cs` ‚Äì Represents a product with price, quantity, and total cost.
- `Order.cs` ‚Äì Manages customer‚Äôs products, calculates total, and prints labels.
- `Program.cs` ‚Äì Main entry point with sample orders.

##  Features
- Packing label with product details (name, ID, qty, cost).
- Shipping label with customer name and address.
- Calculates total order cost including shipping:
  - **$5** for USA customers.
  - **$35** for international customers.

##  Concepts Practiced
- Object-Oriented Programming (OOP)
- Encapsulation
- Lists & Iteration
- String formatting
- Conditional logic
